# Stories that Connect

## Doctrine

Read [../storyteller-tactics/references/doctrine.md](../storyteller-tactics/references/doctrine.md). Job: **build empathy** so better ideas, products, and leadership follow.

## Draft-first

Propose a filled [assets/connect-story-brief.md](assets/connect-story-brief.md) from any interview notes or context. Match density/shape of [assets/exemplar-connect-story-brief.md](assets/exemplar-connect-story-brief.md) (fictional — don’t copy its facts). Mark `TBD`. Ask them to react — then run missing listening steps.

## Process

1. **Story Listening** — four passes: emotion → timeline → decisions → expertise. (ch04)
2. **Abstractions** — observe Action → Imitation → Ritual → Narrative → Rationalisation; mine say/do gaps. (ch04)
3. **Universal Stories** — free will, conflict/cooperation, fear/bravery, rites of passage, etc. (ch03)
4. **Story-ish Conversations** — prompt visual, who/what/where, emotion, conflict, change. (ch06)
5. **Circle of Life** — Child/Adult/Parent; conflict, change, or character. (ch03)
6. Bank permission + whose story it is.

## Output

Filled brief + one empathetic narrative told *about* the person (with permission), not *as* them if you lack lived experience.

## Anti-patterns

- Interview-only with no observation
- Stealing someone’s story without asking
- Jumping to pitch before empathy
