# Recipe routing

Use when `storyteller-start` needs to pick a primary skill.

| User signal | Recipe skill | First artifact |
| --- | --- | --- |
| Sell, convert, win the deal, case study for buyers, trust to deliver | `stories-that-sell` | Sales Story Brief |
| Get buy-in, change, leap of faith, rally the team/board behind a plan | `stories-that-motivate` | Motivation Story Brief |
| Explain expertise, research readout, “why listen to me,” data that won’t stick | `stories-that-convince` | Convince Story Brief |
| Empathy, user research, walk in their shoes, discover what people care about | `stories-that-connect` | Connect Story Brief |
| Strategy narrative, “why this direction,” make the plan care-about-able | `stories-that-explain` | Explain Story Brief |
| Team culture, how we work, onboarding stories, learn from setbacks | `stories-that-lead` | Lead Story Brief |
| Presentation, keynote, deck, nail the talk, Show & Tell | `stories-that-impress` | Impress Story Brief |
| Named arc/tactic only (Man in a Hole, POPP, Movie Time…) | `storyteller-tactics` | Apply named tactic; no full recipe |
| Vague “help me tell a story” / “where do I start” | Diagnose job → one recipe above | Filled step-1 of that recipe |

## Internal diagnostic (do not quiz cold)

If the job is still unclear after a draft proposal, quietly check which gap is largest:

1. Why need a story? → Concept (often Motivate / Explain)
2. Where to find it? → Explore (often Connect / Convince)
3. Your role? → Character (Trust Me / Hero & Guide)
4. What must it do? → Function (Sell / Pitch / Impress)
5. How to plan? → Structure arcs
6. How to tell? → Style
7. How to share? → Organise / Story Bank

Route to the **recipe** that closes the job, not the category name alone.

## Sequence defaults

After the primary recipe draft:

1. Movie-check + Three is the Magic Number
2. Audience Profile gaps → ask only those
3. Log Story Bank metadata (whose story, permission, audience)
4. Optional: hand off to `storyteller-tactics` for a deeper arc swap
