- Raw material on hand (facts, anecdotes, data, quotes): Northline gate-time before/after; one supervisor quote; peer name-drop (Raj) with limited permission.
- Whose story is it / permission: Northline OK with written permission; Raj quote private rooms only.
- Primary skill to run next: stories-that-sell
