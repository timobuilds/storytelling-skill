# Chapter 8: Style Tactics

## Core Idea
Style tactics answer *how to tell the story*: make it visual, memorable, vivid, mysterious, and well-synced with visuals.

## Frameworks Introduced

- **Movie Time** (Desert Island): Brains run visual simulations; abstracts don't play a movie.
  1. Find a movie moment for the key point (What's It About? if stuck).
  2. **Action** (where, who, what happens) + **Emotion** (stakes, feel) → then **Meaning** (change, ask).
  3. Movie-check writing: end of paragraph → "what's the movie now?" If none, you're not storytelling.

- **Story Hooks** (partial card): Grab attention without losing sense.
  - **Unexpected**: zag when others zig (e.g. "Why you'll fail" as a succeed story).
  - **Knowledge**: rules, explainers, listicles, life-hacks, secrets → expertise + insider status.

- **Three is the Magic Number**: Limit what you want remembered to three; use patterns —
  - **Attention Three** (Bond): once / twice / third = deliberate.
  - **Reversal Three** (Little Pigs): setup, setup, reversal.
  - **Moderate Three** (Goldilocks): extreme, extreme, just right.

- **Rolls Royce Moment** (Ogilvy): One vivid, exemplary, relatable detail that stands for the whole ("loudest noise… electric clock"). Do the homework; pair with Secrets & Puzzles / Data Detectives.

- **Leave It Out! / Mind the Gap**: Mystery by omission (Hemingway six-word story; unseen dragon). Hint at gaps so audience co-authors — then sense-check their fill-ins.

- **Show & Tell**: Presentation as a washing line.
  - Boring: script duplicates slides.
  - Confusing: script drifts from image.
  - Engaging: show image → name it → tell what they *can't* see → new image → re-peg attention.

## Key Concepts
- **Movie moment**: Sensory scene that embodies an abstract idea.
- **Rolls Royce moment**: Single exemplary detail that proves the whole.
- **Washing-line Show & Tell**: Peg attention back to visuals rhythmically.

## Mental Models
- Use Movie Time whenever language goes abstract (sustainability, accessibility, strategy…).
- Use Three is the Magic Number for research readouts and pitches.
- Use Rolls Royce Moment for brand/product proof points.
- Use Show & Tell for every slide deck.
- Use Leave It Out to freshen stale decks and create intrigue.

## Anti-patterns
- **Reading slides aloud**.
- **More than three core takeaways**.
- **Abstract claims with no sensory scene**.
- **Gaps that invite wrong assumptions** without sense-check.

## Key Takeaways
1. If there's no movie in the head, there's no story yet.
2. Three patterns beat long lists for memory.
3. One vivid exemplary detail > a page of specs.
4. Mystery invites co-authorship — verify the fill-in.
5. Desert Island Style card: **Movie Time**.
