# Cheatsheet — Storyteller Tactics

## Diagnose (Story Building System)
| Stuck on… | Go to |
|-----------|-------|
| Why need a story? | **Concept** |
| Where to find it? | **Explore** |
| Your role? | **Character** |
| What it must do? | **Function** |
| How to plan? | **Structure** |
| How to tell? | **Style** |
| How to share? | **Organise** |

## Desert Island 7
1. Dragon & the City → 2. Story Listening → 3. Trust Me, I'm an Expert → 4. Pitch Perfect → 5. Man in a Hole → 6. Movie Time → 7. Story Bank

## Recipes (one-liners)
| Goal | Stack |
|------|-------|
| Sell | Audience Profile → Sales Stories → Social Proof → Rags → Pitch |
| Motivate | Dragon → Drive → Conflicts → Curve → No Easy Way |
| Convince | Threes → Funny → Data → Trust Me → Hero & Guide |
| Connect | Listen → Abstractions → Universal → Story-ish → Circle |
| Explain | Order/Chaos → Good/Evil → What's It About → Rolls → Hooks |
| Lead | Curious → Man in Hole → Dashboard → Failures → Bank |
| Impress | Movie → Five Ts → Hooks → Show & Tell → Cut to Chase |

## Pitch: POPP
**P**roblem (−) → **O**pportunity (+) → **P**ractical Steps (−) → **P**romise (+)

## Arc picker
| Need | Arc |
|------|-----|
| Lesson / recovery | Man in a Hole |
| Optimistic value | Rags to Riches |
| Realistic belief | No Easy Way |
| Pre-empt sceptics | Pride & Fall → No Easy Way |
| Adventure/change | Voyage & Return |
| On the spot | Five Ts |

## Innovation Curve — what to tell
| Who | Story focus |
|-----|-------------|
| Pioneers | Process, breakthroughs, prototypes |
| Early mainstream | Tested niche lead |
| Late mainstream | Ready for mass |
| Traditional | Everyone's doing it; easy |

## Style quick checks
- Movie in their head? If no → Movie Time
- More than 3 takeaways? → Three is the Magic Number
- One vivid proof detail? → Rolls Royce Moment
- Slides = script? → Show & Tell washing line
- Room going cold? → Cut to the Chase

## Organise quick checks
- Permission logged? → Story Bank
- Right cell? → Big/Small × Inside/Outside
- Know hopes & fears? → Audience Profile
